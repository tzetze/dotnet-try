# Project guidelines

## Required skills

This project relies on the `conventional-commit` and `caveman` skills. At the
start of a session, check that both are available. If either is missing, tell
the user which one and ask whether to install it before continuing.

## Commits

Use the `conventional-commit` skill for all commit messages in this repo.

## Communication

Use the `caveman` skill — respond in caveman mode.
